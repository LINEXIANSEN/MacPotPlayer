# 2026-04-08 工作日志

## MacPotPlayer 项目

完成了 Mac 视频播放器完整代码框架（方案A：Swift + FFmpeg + AVFoundation + Metal）。

### 已完成模块
1. **项目结构** - Package.swift、project.yml（XcodeGen）、完整目录
2. **播放引擎** - FFmpegPlayer.swift（解码框架）、PlayerManager.swift（单例控制器）
3. **主界面** - ContentView.swift、VideoPlayerView.swift（Metal GPU渲染）
4. **控制栏** - PlayerControlsView.swift（进度条/音量/速度/全屏）
5. **字幕系统** - SubtitleManager.swift（SRT/ASS/VTT解析）、SubtitleOverlayView.swift
6. **播放列表** - PlaylistManager.swift、PlaylistSidebarView.swift（侧边栏）
7. **书签** - BookmarkManager.swift、BookmarkPanelView.swift
8. **均衡器** - AudioProcessingEngine.swift（10段EQ+10预设）、EqualizerView.swift
9. **截图** - ScreenshotManager.swift（PNG/JPEG/BMP）
10. **画面调节** - VideoAdjustmentsView.swift（亮度/对比度/饱和度/AB循环）
11. **设置** - PreferencesManager.swift、PreferencesView.swift（7个标签页）
12. **桥接层** - FFmpegBridge.h/.mm（ObjC++封装FFmpeg）、ASSBridge.h/.mm（libass）
13. **工具类** - SupportedFormats、PlaybackProgressStore、MediaKeyHandler、Logger
14. **资源** - Info.plist、.entitlements、setup_dependencies.sh、README.md

## 新增：360° 全景视频支持（VR 模式）

新增了 3 个文件，修改了 4 个文件：

- **新增** `VRCameraController.swift` - 视角控制器（偏航/俯仰/FOV，鼠标拖拽+滚轮缩放+触摸板捏合）
- **新增** `VRPlayerView.swift` - Metal 球面投影渲染器（equirectangular → 球面内壁映射，64x64 网格）
- **修改** `VideoPlayerView.swift` - 重构为 View 容器，根据 `isVRMode` 自动切换普通/VR 渲染，含 VR HUD 叠加层
- **修改** `PlayerManager.swift` - 添加 `isVRMode`、`vrCamera`、全景视频自动检测（元数据+文件名+2:1宽高比）
- **修改** `PlayerControlsView.swift` - 添加 360° VR 切换按钮（蓝色高亮激活状态）
- **修改** `FFmpegPlayer.swift` - `MediaInfo` 新增 `videoWidth/videoHeight` 字段
- **新增** `.github/workflows/build.yml` - PR/push 时自动在 macOS 14 构建，上传 .app artifact
- **新增** `.github/workflows/release.yml` - 打标签 v*.*.* 时自动构建 Release + 打包 .dmg + 发布 GitHub Release
- **新增** `Scripts/build_dmg.sh` - 本地 dmg 打包辅助脚本
- **新增** `.gitignore` - 过滤构建产物、IDE 文件
